# VelocityOS Feature Documentation

## Landing Page

- Startup-style marketing route
- Hero, feature framing, pricing section, and workspace CTA
- Lightweight bundle for fast public entry

## Velocity Mode

- Replaces the old circular timer
- Uses a supercar-style speedometer UI
- Shows Focus Speed, RPM, Turbo, and Nitro
- Supports focus, short break, and long break modes

## Garage System

- XP-based progression
- 8 unlockable vehicles
- Vehicle stats, descriptions, and completion percentage

## Reward System

- Enhanced completion overlay
- XP summary
- streak summary
- next unlock display
- vehicle unlock callout
- motivational quote

## AI Driver Coach

- Daily and weekly interpretation of session data
- Productive window and strongest day summary
- Simple recommendation engine

## Social and Cloud Foundations

- Social page with leaderboards/focus-room preview
- Google-login preview state
- Supabase client scaffold ready for real auth/sync hookup

## Settings and Themes

- Theme marketplace
- Background effect selection
- Timer duration controls
- Ambient sound controls
